# 🐧 __Penguin

Application for driving a small TFT 3.5 inch 480x320 display with vid 0x1A86 && pid == 0x5722.

using a reverse-engineered communication protocol.

This project demonstrates how to control an embedded display device, send raw frame data, and render animations (like a dancing penguin 🐧) over a custom transport layer.

---

## ✨ Overview

**__Penguin** is a lightweight C/C++ winapi application that communicates with a TFT screen through a custom protocol.

It allows you to:
- Initialize and configure the display
- Send raw image frames
- Render animations in real-time
- Control orientation (portrait / landscape)
- Work with reverse-engineered packet structures
